Lyricus preferences

Site priorities

Lyricus fetches its lyrics from multiple sources. To select which sites to use, simply check the checkbox next to a site's name.
To change the order in which the sites are queried, drag the site names in the order you want them.

Lyric display is always on top

If this setting is activated, the lyric window will always appear above all other windows, even when Lyricus is not active.

Don't warn about index age

By default, Lyricus will warn when your reverse lyric search index is over one week old. Use this option to disable the warning.

Text and background colors

You can change the text and background colors used in the main window.
It's recommended that the edit mode background color is slightly different from the standard background color to make it easy to see when edit mode is active.
For more information on edit mode, see editing.
